PROJECT OVERVIEW
===============================================================================

This is a professional-grade interactive presentation analyzing the competitive
landscape for "Pistachio Dream" — a premium pistachio white chocolate oat milk
drink with tapioca pearls and coffee.

Created for: University-level Business Strategy course
Technology: HTML5, CSS3, JavaScript (Reveal.js framework, Chart.js)
Design: Premium beverage brand aesthetic with sophisticated animations

===============================================================================
STRATEGIC POSITIONING SUMMARY
===============================================================================

CHOSEN POSITION: Premium Accessible Innovation

RETAIL PRICE: £4.49 (400ml bottle)
PRICE PER 100ML: £1.14

JUSTIFICATION:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Cost-Plus Foundation
   - Premium ingredients (oat milk, tapioca pearls, pistachio extract, 
     white chocolate) require higher COGS than standard RTD beverages
   - Cold chain logistics and specialized packaging add to production costs
   - Quality tapioca pearls require sophisticated preservation technology

2. Competitive Anchoring
   - Positioned between Boba RTD (£4.49, £1.00/100ml) and 
     Oatly Cold Brew (£4.29, £1.30/100ml)
   - Matches Boba RTD in absolute price but offers superior sophistication
   - Undercuts Oatly on per-ml basis while adding texture innovation

3. Psychological Pricing
   - £4.49 stays below the £5.00 "luxury barrier" for impulse purchases
   - Ends in .49 to maximize perceived value vs. round numbers
   - Price signals "premium but accessible" rather than "exclusive luxury"

4. Value Perception
   - Comparable to a single specialty café beverage (£5-7)
   - Offers convenience (grab-and-go) + shelf life café drinks lack
   - Multi-sensory experience (flavor + texture + function) justifies premium
   - Plant-based positioning appeals to sustainability-conscious consumers
     willing to pay 15-30% more

5. Category Creation Strategy
   - First-mover advantage in "premium plant-based bubble tea coffee" category
   - Innovation premium justified: no direct substitute exists
   - Price establishes category expectation for future competitors

STRATEGIC WHITE SPACE:
We occupy the intersection of four growing markets (bubble tea, plant-based
milk, premium RTD coffee, functional beverages) at a price point that signals
quality without exclusivity — maximizing market accessibility while maintaining
premium brand perception.

HOW TO RUN THE PRESENTATION
===============================================================================

METHOD 1: SIMPLE BROWSER OPEN (Recommended)

2. Double-click on "index.html"
3. Your default web browser will open the presentation
4. Use arrow keys or space bar to navigate

METHOD 2: LOCAL WEB SERVER (For Best Results)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Some features work better with a local server:

Using Python 3:
  cd presentation
  python -m http.server 8000
  
  Then open: http://localhost:8000

Using Python 2:
  cd presentation
  python -m SimpleHTTPServer 8000
  
  Then open: http://localhost:8000

Using Node.js (if installed):
  cd presentation
  npx http-server -p 8000
  
  Then open: http://localhost:8000

Using VS Code:
  1. Install "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

===============================================================================
TROUBLESHOOTING
===============================================================================

ISSUE: Presentation won't load
  → Check internet connection (required for CDN resources)
  → Try a different browser
  → Clear browser cache

ISSUE: Animations not working
  → Ensure JavaScript is enabled
  → Update to latest browser version
  → Check console for errors (F12 → Console)

ISSUE: Chart not displaying
  → Verify internet connection (Chart.js loads from CDN)
  → Navigate away and back to chart slide to re-trigger
  → Check browser console for errors

ISSUE: Fonts look different
  → Google Fonts may be blocked (firewall/network)
  → System will fall back to default fonts
  → Download fonts locally if needed
